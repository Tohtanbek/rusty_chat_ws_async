
# Rust async websocket chat


## Description

In the current state of development it is possible to see online users, choose chat room, and.. chat :)


The message history is saved in local cache
## Stack

- Simple GUI written fully in rust using [iced](https://docs.rs/iced/latest/iced/) crate
- Server and client use [tokio](https://docs.rs/tokio/latest/tokio/) for async operations and [tokio-tungestine](https://docs.rs/tokio-tungstenite/latest/tokio_tungstenite/) for websocket implementation
